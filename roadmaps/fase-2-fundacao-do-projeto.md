# Fase 2 - Fundacao do Projeto

## Objetivo
Estabelecer a base tecnica do frontend com estrutura limpa, design tokens e layout inicial pronto para evolucao.

## Escopo
- Bootstrap do projeto
- Setup de estilo e tokens
- Organizacao de pastas
- Layout estrutural

## Estrutura Tecnica
1. Inicializar Vite + React + TypeScript.
2. Configurar Tailwind e arquivo de tokens CSS.
3. Definir aliases e convencoes de imports.
4. Preparar componentes base (Button, Section, Container, Heading).

## Estrutura de Pastas (proposta)
- src/app
- src/components
- src/sections
- src/lib
- src/styles
- src/assets

## Design Tokens Iniciais
- Cores semanticas (bg, surface, text, primary, accent)
- Escala tipografica
- Escala de espacamento
- Radius/sombra/glow

## Entregaveis
- Projeto executando localmente
- Layout base com Header, secoes placeholder e Footer
- Sistema inicial de tokens aplicado

## Criterios de Aceite
- Build sem erros
- Estrutura consistente e documentada
- Base visual aplicavel em todas as secoes

## Riscos
- Setup crescer demais no inicio
  - Mitigacao: manter o minimo para V1.

## Checklist
- [ ] Projeto inicializado
- [ ] Tailwind configurado
- [ ] Tokens criados
- [ ] Layout base pronto
- [ ] Convencoes documentadas
