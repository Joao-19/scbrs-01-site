# Workflow and Commands

## Install
From repository root:

```powershell
npm install
```

## Development
```powershell
npm run dev
```

## Build
```powershell
npm run build
```

## Preview
```powershell
npm run preview
```

## Direct frontend commands
- `npm run dev:front`
- `npm run build:front`
- `npm run preview:front`

## Lint
```powershell
npm --prefix frontend run lint
```
